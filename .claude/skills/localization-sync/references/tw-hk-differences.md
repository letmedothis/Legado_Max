## 字符级别差异

以下汉字的选用在两个区域有差异，需注意：

| 概念 | 台湾常用 | 香港常用 |
|------|---------|---------|
| 为 / 爲 | 為 (U+70BA) | 爲 (U+7232) |
| 读 / 閲 | 讀 (U+8B80) | 閲 (U+95B2) |
| 线 / 綫 | 線 (U+7DDA) | 綫 (U+7DAB) |
| 着 / 著 | 著 (U+8457) | 着 (U+7740) |
| 里 / 裏 | 裡 (U+88E1) | 裏 (U+88CF) |
| 台 / 臺 | 臺 (U+81FA) 正式语境 | 台 (U+53F0) 较常见 |

## 翻译原则

1. **优先参考已有翻译** — 查看同一文件中相似概念的翻译用词，保持一致
2. **当不确定时** — 优先使用该区域更通用的词汇，并在修改说明中标注
3. **技术术语** — 台湾偏好意译（如「軟體」），香港受内地影响较多用「件」结尾（如「軟件」）
4. **保持统一** — 同一文件内的用词风格必须一致，不可混用
