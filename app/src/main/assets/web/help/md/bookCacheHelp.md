# 书籍缓存备份机制

## 备份方式

书籍缓存备份会将您选中的书籍缓存打包成一个 ZIP 文件,文件名格式为 `书籍缓存yyyyMMdd_HHmmss.zip`。

## 备份内容

备份 ZIP 文件包含以下内容：

| 文件/目录 | 说明 |
|----------|------|
| `book_cache/` | 缓存文件目录，包含所有选中书籍的章节缓存文件（.nb 格式） |
| `bookCacheIndex.json` | 书籍缓存索引，记录每本书的缓存文件映射关系 |
| `bookCacheBooks.json` | 书籍信息列表，包含书名、作者、来源等基本信息 |
| `bookChapterCache.json` | 章节目录列表，包含每本书的章节标题和顺序 |

## 恢复说明

恢复备份时：

1. **书籍信息恢复**：如果书架上没有该书，会自动添加到书架
2. **章节目录恢复**：恢复书籍的章节目录数据
3. **缓存文件恢复**：将缓存文件恢复到对应目录，恢复后可直接离线阅读

## 关于书源

**备份不包含书源信息**。恢复后：

- 如果您的书架上有该书对应的书源（通过 bookUrl 匹配），阅读时会自动使用该书源
- 如果没有对应书源，书籍仍可正常阅读已缓存的章节，但无法更新或加载新章节
- 建议在恢复前先备份书源，或确保书源已同步

## 注意事项

- 缓存文件可能较大，请确保有足够的存储空间
- 恢复时如果书籍已存在，会跳过该书的信息恢复，但仍会恢复缓存文件
