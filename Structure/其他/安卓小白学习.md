# Android 小白学习路线

## 一、语言基础（2-3周）

### Kotlin 基础语法
- 变量与数据类型（val、var、基本类型）
- 控制流（if、when、for、while）
- 函数与Lambda表达式
- 面向对象编程（类、对象、接口、继承）
- 空安全机制（?.、?:、!!）

### Kotlin 进阶特性
- 扩展函数与属性
- 高阶函数与函数式编程
- 集合操作（list、map、filter、fold）
- 协程基础（launch、async、suspend）

**学习资源：**
- Kotlin 官方文档：https://kotlinlang.org/docs/
- Kotlin Koans 在线练习
- 《Kotlin实战》书籍

**实践项目：**
- 编写一个控制台计算器
- 实现一个简单的待办事项列表（控制台版本）

---

## 二、环境搭建（1周）

### 开发工具安装
- 安装 JDK 17 或更高版本
- 下载并安装 Android Studio（最新稳定版）
- 配置 Android SDK、模拟器
- 熟悉 Android Studio 界面与快捷键

### 项目结构理解
- Gradle 构建系统基础
- 项目目录结构（app、build.gradle、manifests）
- 资源文件管理（res 目录下的 layout、drawable、values）

**实践任务：**
- 创建第一个 Android 项目
- 在模拟器上运行 Hello World
- 尝试修改应用名称和图标

---

## 三、Android 平台基础（4-6周）

### 3.1 核心组件
- **Activity**：生命周期、启动模式、Intent 传递数据
- **Fragment**：生命周期、动态加载、ViewModel 集成
- **Service**：启动方式、前台服务、后台任务
- **BroadcastReceiver**：静态/动态注册、系统广播
- **ContentProvider**：数据共享基础

### 3.2 用户界面开发
- 常用布局（LinearLayout、ConstraintLayout、FrameLayout）
- 常用控件（TextView、Button、EditText、RecyclerView）
- 自定义 View 基础
- Material Design 组件使用
- 主题与样式定制

### 3.3 数据存储
- SharedPreferences 轻量数据存储
- Room 数据库（Entity、DAO、Database）
- 文件存储（内部/外部存储）
- DataStore 使用

### 3.4 网络通信
- Retrofit + OkHttp 网络请求
- JSON 解析（Gson、Moshi）
- 协程网络请求处理
- 错误处理与异常管理

**学习资源：**
- Android 官方文档：https://developer.android.com
- Google Codelabs 实践教程
- 《第一行代码 Android》

**实践项目：**
- 实现一个天气查询应用（网络请求 + UI展示）
- 开发一个记事本应用（Room 数据库 + RecyclerView）

---

## 四、进阶技能（4-6周）

### 4.1 架构模式
- MVVM 架构模式
- Repository 模式
- UseCase 模式
- Clean Architecture 基础

### 4.2 Jetpack 组件
- **ViewModel**：数据管理与生命周期感知
- **LiveData**：响应式数据更新
- **Navigation**：导航组件
- **Paging**：分页加载
- **WorkManager**：后台任务调度

### 4.3 Kotlin 协程与 Flow
- 协程作用域（CoroutineScope）
- Flow 与 StateFlow
- 冷流与热流
- 异常处理与取消

### 4.4 依赖注入
- Hilt 基础使用
- @Inject、@Module、@Provides 注解
- ViewModel 注入

### 4.5 测试
- 单元测试（JUnit、Mockk）
- UI 测试（Espresso）
- 测试覆盖率

**学习资源：**
- Android Jetpack 官方指南
- 《Android 架构组件实战》
- 项目源码学习：legado、legado_Plus

**实践项目：**
- 重构记事本应用为 MVVM 架构
- 添加单元测试和 UI 测试

---

### 5.2 启动优化
- 启动时间分析
- 延迟初始化
- 启动优化技巧

### 5.3 网络优化
- 请求缓存策略
- 图片加载优化（Coil、Glide）
- 网络状态监听

### 5.4 电量优化
- 后台任务优化
- WorkManager 合理使用
- 电量消耗分析

**实践任务：**
- 使用 Profiler 分析应用性能
- 优化应用启动时间
- 实现图片缓存加载

## 八、学习建议

### 学习方法
1. **边学边练**：每学一个知识点就写代码验证
2. **阅读源码**：学习优秀开源项目的设计思路
3. **记录总结**：写博客或笔记记录学习过程
4. **参与社区**：加入 Android 开发者社区交流

### 时间规划
- 每天至少 2 小时学习时间
- 每周完成一个小功能模块
- 每月完成一个完整项目

### 学习资源
- 官方文档（最权威）
- GitHub 开源项目
- 技术博客（Medium、掘金）
- 视频教程（Udacity、Coursera）
